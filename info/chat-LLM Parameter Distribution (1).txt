### USER
I need to distribute a single LLM with 70B parameters between many computers to run it using CPUs only. How can I do this best way?

### ASSISTANT
Distributing a 70B parameter LLM across multiple CPU-only computers is technically possible, but it comes with significant performance caveats. **Network latency is your biggest enemy.** If your computers are connected via standard Wi-Fi or 1GbE Ethernet, the inference speed will likely be unusable (less than 1 token per second).

However, if you have a wired LAN (10GbE or better) and need to pool RAM because no single machine has enough memory, this is the best approach.

Here is the comprehensive guide to doing this the "best way" using current open-source technology.

---

### 1. The Core Strategy: `llama.cpp` + GGUF + RPC
The industry standard for CPU inference is **`llama.cpp`**. It supports the **GGUF** format (highly quantized models) and has a built-in **RPC (Remote Procedure Call)** feature that allows you to offload model layers to other machines on the network.

*   **Why this stack?** It is optimized for AVX2/AVX512 CPU instructions, minimizes memory usage via quantization, and handles the tensor splitting across the network.
*   **Alternative:** **Petals** (easier setup, works over the internet, but often slower on a local LAN).

### 2. Prerequisites & Hardware Reality Check

Before starting, calculate your memory requirements. A 70B model cannot run in FP16 (140GB RAM) on consumer hardware. You must quantize it.

| Quantization | Size (Approx.) | Min RAM Required | Quality Loss |
| :--- | :--- | :--- | :--- |
| **Q4_K_M** (Recommended) | ~40 GB | 48 GB | Minimal |
| **Q5_K_M** | ~45 GB | 52 GB | Negligible |
| **Q8_0** | ~70 GB | 80 GB | None |

**Network Requirements:**
*   **Wired Connection Only:** Do not use Wi-Fi.
*   **Speed:** 10 Gigabit Ethernet (10GbE) is highly recommended. 1GbE will result in extreme latency.
*   **Topology:** All worker nodes should connect to a central switch, not daisy-chained.

### 3. Step-by-Step Implementation

#### Step A: Prepare the Model
You need the model in **GGUF** format. Do not use PyTorch (`.pt`) or Safetensors directly.
1.  Download a 70B GGUF model (e.g., Llama-3-70B-Instruct) from HuggingFace (look for uploads by *MaziyarPanahi* or *Bartowski*).
2.  Choose **Q4_K_M.gguf** for the best balance of speed and RAM.
3.  Place this file on the **Main Node** (the computer that will coordinate the request).

#### Step B: Install `llama.cpp` on All Nodes
You need to compile `llama.cpp` on every computer involved (Main Node + Worker Nodes).

```bash
# On ALL computers (Linux/Mac/WSL)
git clone https://github.com/ggerganov/llama.cpp.git
cd llama.cpp
make -j $(nproc)  # Compiles with all CPU cores
```
*Note: Ensure you have build essentials (`build-essential`, `cmake`) installed.*

#### Step C: Start the Worker Nodes
On every computer **except** the Main Node, you need to run an RPC server that waits for tensor computations.

```bash
# On Worker Node 1, 2, 3...
./llama-server --host 0.0.0.0 --port 50052 --rpc
```
*   `--host 0.0.0.0`: Allows connections from other computers.
*   `--port 50052`: The port for RPC communication (ensure this port is open in your firewall).
*   `--rpc`: Puts the server into RPC worker mode.

#### Step D: Start the Main Node
On the Main Node, you will launch the actual inference server and tell it where the workers are.

```bash
# On Main Node
./llama-server \
  --model /path/to/llama-3-70b-instruct.Q4_K_M.gguf \
  --host 0.0.0.0 \
  --port 8080 \
  --rpc 192.168.1.101:50052,192.168.1.102:50052,192.168.1.103:50052 \
  -ngl 999 \
  -t 8
```
*   `--rpc`: Comma-separated list of Worker IP addresses and ports.
*   `-ngl 999`: Offload as many layers as possible (since we are using CPU/RPC, this tells it to use the distributed setup).
*   `-t 8`: Number of threads to use on the Main Node's CPU (adjust based on your CPU cores).

#### Step E: Configure Layer Distribution (Optional but Recommended)
By default, `llama.cpp` tries to balance layers automatically. However, if your computers have different amounts of RAM, you may need to manually specify how many layers go to each node to prevent Out-Of-Memory (OOM) crashes.

You can control this by specifying the split in the RPC string or by limiting memory usage on workers. Currently, `llama.cpp` distributes evenly. If one node has 16GB RAM and another has 64GB, the 16GB node will crash.
*   **Solution:** Ensure all nodes have roughly equal available RAM, or only include nodes with sufficient capacity in the `--rpc` list.

### 4. Critical Optimization Tips

1.  **Disable Hyperthreading:** For CPU inference, physical cores are often faster than logical threads. If you have 8 cores / 16 threads, try setting `-t 8` instead of 16.
2.  **NUMA Pinning:** If your Main Node has dual CPUs, ensure the process isn't bouncing between NUMA nodes, which kills memory bandwidth. Use `numactl`.
3.  **Huge Pages:** Enable Linux Huge Pages to reduce TLB misses and improve memory bandwidth.
    ```bash
    sudo sysctl -w vm.nr_hugepages=1024
    ```
4.  **Batch Size:** Keep the prompt processing batch size (`-b`) reasonable. High batch sizes increase RAM usage significantly. Start with `-b 512`.

### 5. Alternative: Petals (Easier, but potentially slower)
If compiling `llama.cpp` and managing RPC ports is too complex, **Petals** is a library designed specifically for distributed inference over the internet or LAN. It works like BitTorrent for LLMs.

*   **Pros:** Handles node discovery, dynamic joining/leaving, and sharding automatically.
*   **Cons:** Higher overhead than `llama.cpp`, Python-based (slower).

**How to use Petals:**
1.  Install: `pip install petals`
2.  Run on all nodes:
    ```python
    from petals import AutoDistributedModel, AutoDistributedTokenizer
    model = AutoDistributedModel.from_pretrained("meta-llama/Meta-Llama-3-70B-Instruct")
    ```
    *Note: Petals usually connects to public swarm. To run a private swarm, you need to set up a DHT (Distributed Hash Table) bootstrap node.*

**Is this actually the best way?**
Probably not, unless you are forced to use existing hardware.

**Why?**
A 70B model at Q4 quantization requires ~40GB of RAM.
*   **Scenario A (Your Plan):** 4 computers with 16GB RAM each. Connected via 1GbE.
    *   *Result:* Network latency dominates. Speed: **0.5 tokens/second**.
*   **Scenario B (Single Machine):** 1 computer with 64GB RAM (DDR5).

**Final Verdict:**
1.  **If you have 10GbE+ Network:** Use **`llama.cpp` with RPC** as described above.
2.  **If you have 1GbE Network:** Do not distribute. Instead, run the model on the single machine with the most RAM and use **OS Swap** (create a large swap file). It will be slow, but network latency across 1GbE is often worse than disk swapping.

    *   *Logic:* Node A tells a public Rendezvous Server "I am at IP:X". Node B asks the server for A's IP. Both send UDP packets to each other simultaneously to open the firewall ports.
    *   *Note:* You need at least one public bootstrap node ( Rendezvous ) to start the handshake. This is not centralized control, just a phonebook.

#### C. Cryptography (Vendored, Not Written)
**Critical Warning:** Do not write your own encryption. It *will* be broken.
*   **Solution:** Vendor **mbedTLS** or **libsodium** source code into `/src/crypto`.
*   **Why:** They are C-based, lightweight, compile with GCC/MSVC easily, and are open source.
*   **Usage:**
    *   **TLS 1.3:** Encrypt all traffic between nodes (prevent ISP throttling or eavesdropping).
    *   **Ed25519:** Each node generates a keypair on first run. This is their **Identity**.
    *   **Signing:** Every inference result is signed by the node's private key. This prevents spoofing.

#### D. Discovery (DHT - Distributed Hash Table)
Implement a simplified **Kademlia** DHT in pure C++.
*   **Purpose:** Find nodes that hold specific model layers.
*   **Key:** Hash of the Model Layer ID (e.g., `SHA256("Qwen2.5-3B-Layer-5")`).
*   **Value:** IP:Port of nodes holding that layer.
*   **Logic:** Each node stores a subset of the network map. When you need Layer 5, you query the DHT, which routes you to the node responsible for that key.

#### E. Consensus & Judge (The "Committee")
This is your unique logic layer.

1.  **Request Broadcast:** User sends prompt to 5 random nodes holding the full model (or shards).
2.  **Parallel Inference:** All 5 nodes generate a response.
3.  **Hash Voting:** Nodes broadcast the **hash** of their generated tokens (not the full text yet) to save bandwidth.
4.  **Agreement:** If 3/5 hashes match, the text is requested from one of them.
5.  **Judge Verification:** The local "Judge" model (0.5B) scans the agreed text for safety violations before displaying it to the user.

### 3. Memory Management Strategy (For 4GB Nodes)
To ensure stability on low-RAM nodes:

*   **Custom Allocator:** Implement a `MemoryPool` class in C++.
    *   Pre-allocate a fixed block (e.g., 2.5GB) at startup.
    *   If allocation fails, reject new tasks gracefully instead of crashing.
*   **KV Cache Quantization:** Force `ggml_context` to use `Q8_0` or `Q4_0` for the KV cache.
*   **Swap Handling:** Detect OS swap usage. If swap is active, pause inference to prevent system freeze.

### 4. Security & Trust Model
Since there is no central owner:

*   **Reputation System:** Store a local `reputation.db` (SQLite or simple binary file).
    *   If a node provides a bad hash (mismatched consensus), decrease its reputation.
    *   If reputation < threshold, ban the node's Public Key.
*   **Weight Verification:** When downloading model shards, verify against a known good **SHA256 hash** hardcoded in the source code. This prevents malicious nodes from serving poisoned weights.

### 4. Socket Source (`src/network/Socket.cpp`)

```cpp
#include "network/Socket.hpp"
#include <cstring>
#include <iostream>

namespace FreeAI {
    namespace Network {

        Socket::Socket() : m_handle(InvalidSocket) {}

        Socket::~Socket() {
            Close();
        }

        bool Socket::CreateTCP() {
            m_handle = socket(AF_INET, SOCK_STREAM, IPPROTO_TCP);
            if (m_handle == InvalidSocket) {
                std::cerr << "[SOCKET] Failed to create socket: " << GetLastError() << std::endl;
                return false;
            }
            return true;
        }

        bool Socket::Connect(const std::string& host, int port) {
            if (!IsValid()) return false;

            sockaddr_in serverAddr;
            serverAddr.sin_family = AF_INET;
            serverAddr.sin_port = htons(static_cast<uint16_t>(port));

            // Convert IP string to binary
            if (inet_pton(AF_INET, host.c_str(), &serverAddr.sin_addr) <= 0) {
                std::cerr << "[SOCKET] Invalid address: " << host << std::endl;
                return false;
            }

            int result = connect(m_handle, (struct sockaddr*)&serverAddr, sizeof(serverAddr));
            if (result < 0) {
                std::cerr << "[SOCKET] Connection failed: " << GetLastError() << std::endl;
                return false;
            }
            return true;
        }

        bool Socket::Bind(int port) {
            if (!IsValid()) return false;

            sockaddr_in serverAddr;
            std::memset(&serverAddr, 0, sizeof(serverAddr));
            serverAddr.sin_family = AF_INET;
            serverAddr.sin_addr.s_addr = INADDR_ANY; // Listen on all interfaces
            serverAddr.sin_port = htons(static_cast<uint16_t>(port));

            int result = bind(m_handle, (struct sockaddr*)&serverAddr, sizeof(serverAddr));
            if (result < 0) {
                std::cerr << "[SOCKET] Bind failed on port " << port << ": " << GetLastError() << std::endl;
                return false;
            }
            return true;
        }

        int Socket::Send(const void* data, size_t length) {
            if (!IsValid()) return -1;
#ifdef _WIN32
            return send(m_handle, static_cast<const char*>(data), static_cast<int>(length), 0);
#else
            return send(m_handle, data, length, 0);
#endif
        }

        int Socket::Receive(void* buffer, size_t length) {
            if (!IsValid()) return -1;
#ifdef _WIN32
            return recv(m_handle, static_cast<char*>(buffer), static_cast<int>(length), 0);
#else
            return recv(m_handle, buffer, length, 0);
#endif
        }

        void Socket::Close() {
            if (IsValid()) {
#ifdef _WIN32
                closesocket(m_handle);
#else
                close(m_handle);
#endif
                m_handle = InvalidSocket;
            }
        }

        bool Socket::IsValid() const {
            return m_handle != InvalidSocket;
        }

        std::string Socket::GetLastError() {
#ifdef _WIN32
            int err = WSAGetLastError();
            return "Winsock Error " + std::to_string(err);
#else
            return "POSIX Error " + std::to_string(errno);
#endif
        }

    }
}
```

### Architectural Note: The "Bootstrap" Paradox
To punch a hole, two peers need to know each other's public IP:Port. But they are behind NATs. They need a **public rendezvous point** to introduce them.
*   **Centralized Risk:** If only *one* bootstrap server exists, it is a single point of failure.
*   **Our Solution:** The **Bootstrap Server software** must be open-source (part of this repo). Anyone can host a bootstrap node. The client code should support a *list* of bootstrap nodes.
*   **Privacy:** The bootstrap server **only** exchanges IP addresses. It never sees inference data or model weights. It is a phonebook, not a spy.

```cpp
            if (bind(m_handle, (struct sockaddr*)&serverAddr, sizeof(serverAddr)) < 0) {
                std::cerr << "[UDP] Bind failed: " << Socket::GetLastError() << std::endl;
                return false;
            }
            return true;
        }

        int UDPSocket::SendTo(const void* data, size_t length, const std::string& ip, int port) {
            if (!IsValid()) return -1;

            sockaddr_in targetAddr;
            std::memset(&targetAddr, 0, sizeof(targetAddr));
            targetAddr.sin_family = AF_INET;
            targetAddr.sin_port = htons(static_cast<uint16_t>(port));
            if (inet_pton(AF_INET, ip.c_str(), &targetAddr.sin_addr) <= 0) {
                return -1;
            }

#ifdef _WIN32
            return sendto(m_handle, static_cast<const char*>(data), static_cast<int>(length), 0, (sockaddr*)&targetAddr, sizeof(targetAddr));
#else
            return sendto(m_handle, data, length, 0, (sockaddr*)&targetAddr, sizeof(targetAddr));
#endif
        }

        int UDPSocket::ReceiveFrom(void* buffer, size_t length, std::string& out_ip, int& out_port) {
            if (!IsValid()) return -1;

            sockaddr_in clientAddr;
            socklen_t addrLen = sizeof(clientAddr);

#ifdef _WIN32
            int bytes = recvfrom(m_handle, static_cast<char*>(buffer), static_cast<int>(length), 0, (sockaddr*)&clientAddr, &addrLen);
#else
            int bytes = recvfrom(m_handle, buffer, length, 0, (sockaddr*)&clientAddr, &addrLen);
#endif

            if (bytes > 0) {
                char ipStr[INET_ADDRSTRLEN];
                inet_ntop(AF_INET, &clientAddr.sin_addr, ipStr, INET_ADDRSTRLEN);
                out_ip = std::string(ipStr);
                out_port = ntohs(clientAddr.sin_port);
            }
            return bytes;
        }

        void UDPSocket::Close() {
            if (IsValid()) {
#ifdef _WIN32
                closesocket(m_handle);
#else
                close(m_handle);
#endif
                m_handle = InvalidSocket;
            }
        }

        bool UDPSocket::IsValid() const {
            return m_handle != InvalidSocket;
        }

        bool UDPSocket::SetNonBlocking(bool enable) {
            if (!IsValid()) return false;
#ifdef _WIN32
            u_long mode = enable ? 1 : 0;
            return ioctlsocket(m_handle, FIONBIO, &mode) == 0;
#else
            int flags = fcntl(m_handle, F_GETFL, 0);
            if (flags == -1) return false;
            flags = enable ? (flags | O_NONBLOCK) : (flags & ~O_NONBLOCK);
            return fcntl(m_handle, F_SETFL, flags) == 0;
#endif
        }

    }
}
```

```cpp
namespace FreeAI {

        // Magic number to identify FREE-AI packets

            uint32_t magic;
            uint8_t type;
            uint8_t reserved[3];
            uint32_t payload_size;
```

This is the **Rendezvous Point**. It must be compiled as a separate executable. It should be hosted on a public VPS (or any machine with a public IP).

```cpp
#include <iostream>
#include <unordered_map>
#include <string>
#include "network/NetworkInit.hpp"
#include "network/UDPSocket.hpp"
#include "network/Protocol.hpp"

using namespace FreeAI::Network;

// Simple Peer Record
struct PeerInfo {
    std::string ip;
    int port;
    std::string peer_id; // In real impl, this is Public Key
};

std::unordered_map<std::string, PeerInfo> peers;

int main() {
    std::cout << "[BOOTSTRAP] Starting Free-AI Bootstrap Server..." << std::endl;
    
    if (!NetworkEnvironment::Initialize()) return 1;

    UDPSocket server;
    if (!server.Create()) return 1;
    if (!server.Bind(9090)) return 1; // Standard bootstrap port
    std::cout << "[BOOTSTRAP] Listening on UDP :9090" << std::endl;

    char buffer[1024];
    std::string sender_ip;
    int sender_port;

    while (true) {
        int bytes = server.ReceiveFrom(buffer, sizeof(buffer), sender_ip, sender_port);
        if (bytes <= 0) continue;

        // Basic validation
        if (bytes < sizeof(PacketHeader)) continue;
        PacketHeader* hdr = (PacketHeader*)buffer;
        if (hdr->magic != MAGIC_NUMBER) continue;

        // Handle Packet Types
        if (hdr->type == PT_REGISTER) {
            std::string peer_id = std::string(buffer + sizeof(PacketHeader), hdr->payload_size);
            peers[peer_id] = { sender_ip, sender_port, peer_id };
            std::cout << "[BOOTSTRAP] Registered Peer: " << peer_id << " @ " << sender_ip << ":" << sender_port << std::endl;
        } 
        else if (hdr->type == PT_INTRO_REQUEST) {
            std::string target_id = std::string(buffer + sizeof(PacketHeader), hdr->payload_size);
            if (peers.find(target_id) != peers.end()) {
                PeerInfo& target = peers[target_id];
                // Send target IP back to sender
                std::string response_data = target.ip + ":" + std::to_string(target.port);
                PacketHeader resp_hdr = { MAGIC_NUMBER, PT_INTRO_RESPONSE, {0,0,0}, (uint32_t)response_data.size() };
                
                char send_buf[1024];
                memcpy(send_buf, &resp_hdr, sizeof(resp_hdr));
                memcpy(send_buf + sizeof(resp_hdr), response_data.c_str(), response_data.size());
                
                server.SendTo(send_buf, sizeof(resp_hdr) + response_data.size(), sender_ip, sender_port);
                std::cout << "[BOOTSTRAP] Introduced " << sender_ip << " to " << target.ip << std::endl;
            }
        }
    }
}
```

    *   **Super Node:** Public IP (or successful UPnP). Listens for bootstrap requests and helps others connect.
*   **Dynamic List:** Every node maintains a list of known **Super Nodes**. When you connect to one, it gives you 5 more.

### 2. Configuration Implementation (`src/utils/Config.cpp`)

```cpp
#include "utils/Config.hpp"
#include <algorithm>
#include <cctype>

namespace FreeAI {
    namespace Utils {

        std::string Config::Trim(const std::string& str) {
            size_t start = str.find_first_not_of(" \t\r\n");
            if (start == std::string::npos) return "";
            size_t end = str.find_last_not_of(" \t\r\n");
            return str.substr(start, end - start + 1);
        }

        bool Config::Load(const std::string& filename) {
            std::ifstream file(filename);
            if (!file.is_open()) {
                return false;
            }

            std::string line;
            std::string currentSection;

            while (std::getline(file, line)) {
                line = Trim(line);
                
                // Skip empty lines and comments
                if (line.empty() || line[0] == ';' || line[0] == '#') {
                    continue;
                }

                // Section header [section]
                if (line.front() == '[' && line.back() == ']') {
                    currentSection = line.substr(1, line.size() - 2);
                    continue;
                }

                // Key=Value
                size_t pos = line.find('=');
                if (pos != std::string::npos) {
                    std::string key = Trim(line.substr(0, pos));
                    std::string value = Trim(line.substr(pos + 1));
                    m_data[currentSection][key] = value;
                }
            }

            file.close();
            return true;
        }

        bool Config::Save(const std::string& filename) const {
            std::ofstream file(filename);
            if (!file.is_open()) {
                return false;
            }

            for (const auto& section : m_data) {
                file << "[" << section.first << "]" << std::endl;
                for (const auto& kv : section.second) {
                    file << kv.first << " = " << kv.second << std::endl;
                }
                file << std::endl;
            }

            file.close();
            return true;
        }

        std::string Config::Get(const std::string& section, const std::string& key, const std::string& defaultVal) const {
            auto it = m_data.find(section);
            if (it != m_data.end()) {
                auto it2 = it->second.find(key);
                if (it2 != it->second.end()) {
                    return it2->second;
                }
            }
            return defaultVal;
        }

        int Config::GetInt(const std::string& section, const std::string& key, int defaultVal) const {
            try {
                return std::stoi(Get(section, key, std::to_string(defaultVal)));
            } catch (...) {
                return defaultVal;
            }
        }

        bool Config::GetBool(const std::string& section, const std::string& key, bool defaultVal) const {
            std::string val = Get(section, key, defaultVal ? "true" : "false");
            return (val == "true" || val == "1" || val == "yes");
        }

        void Config::Set(const std::string& section, const std::string& key, const std::string& value) {
            m_data[section][key] = value;
        }

        void Config::SetInt(const std::string& section, const std::string& key, int value) {
            Set(section, key, std::to_string(value));
        }

        void Config::SetBool(const std::string& section, const std::string& key, bool value) {
            Set(section, key, value ? "true" : "false");
        }
```

```cpp
        bool PeerManager::Initialize(const Utils::Config& config) {
            m_listenPort = config.GetInt("network", "bootstrap_port", 9090);
            
            // Load seed nodes from config
            std::string seeds = config.Get("network", "seed_nodes", "");
            std::stringstream ss(seeds);

            while (std::getline(ss, seed, ',')) {
                seed = Utils::Config::Trim(seed);

                }

            // Fallback hardcoded seeds if config is empty

                };
            }

            std::cout << "[PEER] Loaded " << m_seedNodes.size() << " seed nodes." << std::endl;
            return true;

        void PeerManager::Start() {
            m_running = true;

            // Try to bind - determines if we are Super Node
            if (!m_socket.Create()) {
                std::cerr << "[PEER] Failed to create UDP socket." << std::endl;
                return;
            }

            if (m_socket.Bind(m_listenPort)) {
                m_isSuperNode = true;
                m_socket.SetNonBlocking(true);
                std::cout << "[PEER] Running as Super Node (Port " << m_listenPort << " open)." << std::endl;
            } else {
                m_isSuperNode = false;
                std::cout << "[PEER] Running as Leaf Node (NAT detected)." << std::endl;
            }

            m_listenerThread = std::thread(&PeerManager::ListenLoop, this);
            
            // Connect to seeds in background
            std::thread(&PeerManager::ConnectToSeeds, this).detach();
        }

        void PeerManager::Stop() {
            m_running = false;
            if (m_listenerThread.joinable()) {
                m_listenerThread.join();
            }
            m_socket.Close();
        }

        void PeerManager::ListenLoop() {
            char buffer[2048];
            std::string sender_ip;
            int sender_port;

            while (m_running) {
                int bytes = m_socket.ReceiveFrom(buffer, sizeof(buffer), sender_ip, sender_port);
                if (bytes > 0) {
                    HandleBootstrapPacket(m_socket, buffer, bytes, sender_ip, sender_port);
                }
                std::this_thread::sleep_for(std::chrono::milliseconds(10));
            }

        void PeerManager::HandleBootstrapPacket(UDPSocket& sock, char* buffer, int bytes, const std::string& ip, int port) {
            if (bytes < sizeof(PacketHeader)) return;

            PacketHeader* hdr = reinterpret_cast<PacketHeader*>(buffer);
            if (hdr->magic != MAGIC_NUMBER) return;

            // Rate limiting could be added here for security

            if (hdr->type == PT_REGISTER && m_isSuperNode) {
                std::string peer_id = std::string(buffer + sizeof(PacketHeader), hdr->payload_size);
                std::cout << "[PEER] Registered: " << peer_id << " @ " << ip << ":" << port << std::endl;
                
                // Add to known peers
                AddPeer({ip, port, peer_id, std::time(nullptr), true});
            }
            else if (hdr->type == PT_INTRO_REQUEST && m_isSuperNode) {
                // Return a random known peer
                auto peers = GetKnownPeers();
                if (!peers.empty()) {
                    const auto& target = peers[0];
                    std::string response_data = target.ip + ":" + std::to_string(target.port);
                    
                    PacketHeader resp_hdr = { MAGIC_NUMBER, PT_INTRO_RESPONSE, {0,0,0}, static_cast<uint32_t>(response_data.size()) };
                    char send_buf[2048];
                    memcpy(send_buf, &resp_hdr, sizeof(resp_hdr));
                    memcpy(send_buf + sizeof(resp_hdr), response_data.c_str(), response_data.size());
                    
                    sock.SendTo(send_buf, sizeof(resp_hdr) + response_data.size(), ip, port);
                }
            }
            else if (hdr->type == PT_PEER_LIST) {
                // Received peer list from seed, parse and add
                auto newPeers = ParsePeerList(buffer + sizeof(PacketHeader), hdr->payload_size);
                for (const auto& p : newPeers) {
                    AddPeer(p);
                }
                std::cout << "[PEER] Received " << newPeers.size() << " peers from seed." << std::endl;
            }
        }

        void PeerManager::ConnectToSeeds() {
            std::this_thread::sleep_for(std::chrono::seconds(2)); // Wait for socket to be ready

            for (const auto& seed : m_seedNodes) {
                // Parse seed IP:Port
                size_t pos = seed.find(':');
                if (pos == std::string::npos) continue;
                
                std::string ip = seed.substr(0, pos);
                int port = std::stoi(seed.substr(pos + 1));

                std::cout << "[PEER] Connecting to seed: " << ip << ":" << port << std::endl;

                // Send registration
                std::string peer_id = "peer_" + std::to_string(std::time(nullptr)); // Should be Ed25519 pubkey later
                PacketHeader hdr = { MAGIC_NUMBER, PT_REGISTER, {0,0,0}, static_cast<uint32_t>(peer_id.size()) };
                char send_buf[2048];
                memcpy(send_buf, &hdr, sizeof(hdr));
                memcpy(send_buf + sizeof(hdr), peer_id.c_str(), peer_id.size());
                
                m_socket.SendTo(send_buf, sizeof(hdr) + peer_id.size(), ip, port);

                // Request peer list
                PacketHeader listHdr = { MAGIC_NUMBER, PT_PEER_LIST, {0,0,0}, 0 };
                m_socket.SendTo(&listHdr, sizeof(listHdr), ip, port);
            }
        }

        std::vector<PeerInfo> PeerManager::ParsePeerList(const char* data, size_t size) {
            std::vector<PeerInfo> peers;
            std::string str(data, size);
            std::stringstream ss(str);
            std::string line;

            while (std::getline(ss, line, ';')) {
                size_t pos1 = line.find(':');
                size_t pos2 = line.find(':', pos1 + 1);
                if (pos1 != std::string::npos && pos2 != std::string::npos) {
                    PeerInfo p;
                    p.ip = line.substr(0, pos1);
                    p.port = std::stoi(line.substr(pos1 + 1, pos2 - pos1 - 1));
                    p.peer_id = line.substr(pos2 + 1);
                    p.last_seen = std::time(nullptr);
                    p.is_super_node = true;
                    peers.push_back(p);
                }
            }
            return peers;
        }

        std::string PeerManager::BuildPeerList() {
            std::stringstream ss;
            auto peers = GetKnownPeers();
            for (size_t i = 0; i < peers.size() && i < 10; ++i) { // Limit to 10 peers
                if (i > 0) ss << ";";
                ss << peers[i].ip << ":" << peers[i].port << ":" << peers[i].peer_id;
            }
            return ss.str();
        }

        void PeerManager::AddPeer(const PeerInfo& peer) {
            std::lock_guard<std::mutex> lock(m_peerMutex);
            // Check if already exists
            for (const auto& p : m_peers) {
                if (p.ip == peer.ip && p.port == peer.port) {
                    return;
                }
            }
            m_peers.push_back(peer);
        }

        std::vector<PeerInfo> PeerManager::GetKnownPeers() const {
            std::lock_guard<std::mutex> lock(m_peerMutex);
            return m_peers;
        }

        bool PeerManager::IsSuperNode() const {
            return m_isSuperNode;
        }
```
